# COMP3133 Assignment 2 — Employee Management Frontend

Angular frontend for the COMP3133 fullstack application. Utilizes GraphQL API using Apollo Client.

**Woohyuk (Harry) Song | 101524575** 

## Features

- User login and signup
- View all employees
- Add, edit, and delete employees
- View individual employee details

## Backend

GraphQL API: https://comp-3133-101524575-assignment1.vercel.app/

## Tech Stack

- Angular 21
- Apollo Angular + GraphQL

## Running the Program

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

Open [http://localhost:4200](http://localhost:4200) in browser.
